# Patient Appointment Scheduling System

A modern, responsive Patient Appointment Scheduling System built using a custom FIFO (First-In, First-Out) Queue Data Structure.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism design), JavaScript SPA
- **Backend**: Flask (Python)
- **Database**: SQLite (Local) / PostgreSQL (Production)
